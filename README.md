# Linux

### 소유권 관련 명령어
> - **chown**   
>   \- 파일과 디렉토리의 사용자 소유권과 그룹소유권을 변경   
>   \- chown [옵션] 소유자[:그룹명] 파일명   
>   \- 옵션 -R : 하위 디렉토리를 포함하여 모든 파일의 소유권 변경   
> - **chgrp**   
>   \- 파일과 디렉토리의 그룹 소유권을 변경   
>   \- chgrp [옵션] 그룹명 파일명   
>   \- 옵션 -R : 하위 디렉토리를 포함하여 모든 파일의 소유권 변경
